# Create LogsPanel component

Slide-out panel to view service logs (tmux scrollback).

## Design

Files: src/client/components/LogsPanel.tsx, src/client/hooks/useServiceLogs.ts

useServiceLogs hook:
- Fetches logs from /api/projects/:id/services/:sid/logs
- Refetches every 2 seconds while panel is open
- Returns { logs: string, isLoading, error }

LogsPanel component:
- Uses SlideOutPanel wrapper
- Monospace font, dark background
- Auto-scroll to bottom on new content
- Copy 'tmux attach -t mgr-...' button
- Close button

Integrate with ServicesDropdown - clicking View Logs opens this panel.

## Acceptance Criteria

- Panel shows recent log output
- Auto-refreshes every 2 seconds
- Copy attach command works
- Scrolls to bottom on updates
- Can close panel
