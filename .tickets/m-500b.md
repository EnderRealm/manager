# Create ServicesDropdown component

Dropdown in project header showing services with start/stop/restart controls.

## Design

Files: src/client/components/ServicesDropdown.tsx, src/client/components/ProjectView.tsx

ServicesDropdown props:
- projectId: string

Shows button with aggregate status indicator (green/yellow/red/gray dot).
Dropdown lists each service with:
- Status badge (colored dot + text)
- Start button (if stopped/crashed)
- Stop button (if running)
- Restart button (if running)
- View Logs link

Use existing dropdown pattern from FilterDropdown in KanbanBoard.
Add to ProjectView header next to filters.

## Acceptance Criteria

- Dropdown shows all configured services
- Status indicators update in real-time
- Start/stop/restart buttons work
- View Logs link opens logs panel
